## üìã THE PROCESS (10 Minutes Total)

### Phase 1: Setup (30 seconds)
```bash
chmod +x setup.sh && ./setup.sh
```
Makes all deployment scripts executable.

### Phase 2: Deploy to GitHub (1 minute)
```bash
./DEPLOY_NOW.sh
```
Commits and pushes your code to GitHub.

### Phase 3: Deploy to Vercel (5 minutes)
1. Go to https://vercel.com/new
2. Import `visionary-systems/digitaldignity-visualizations`
3. Click "Deploy"
4. Copy your Vercel URL

### Phase 4: Update PHP Site (1 minute)
```bash
./UPDATE_URLS.sh YOUR-URL.vercel.app
```
Automatically updates both PHP files with your Vercel URL.

### Phase 5: Upload to Hostinger (3 minutes)
Upload these 2 files:
- `modules/primer-1.php`
- `assets/js/modules/primer-1.js`
